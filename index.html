I got the idea for this project when I first started seeing D3, a great javascript library for visualizing data, in action and it reminded me of charts I'd seen of advanced hockey stats.

The data is behind drawn from behind the net's player stats page

About these visualizations

These are just recreations of Robert Vollman's player usage charts. The x-axis is Offensive zone start percent, the y-axis is Relative Quality of Competition, and the size of the bubbles reflect each player's relative Corsi. The redder the colour of the bubble the higher the RelC value bubble while lower values result in a yellow bubble. Pure red is 15 and above, pure yellow -15 and lower, and a value of 0 results in an orange bubble.

While these are just recreations, the ultimate purpose was to build a proof of concept for a larger tool that allows users to select from a number of different visualizations and, ultimately, to build customized visualizations by selecting their own data points.

There's still some rough edges around this version as you can see. Overlapping bubbles can cause text to be difficult to read, for which I apologize. I'm looking to guage interest in a tool like this before I spend too much time perfecting it.

Me and advanced stats

Oddly enough, I really don't know much about advanced stats in hockey. In fact, a lot of the concepts are over my head. My primary interest here is experiments in visualization and creating a resource for those who know what to do with this stuff. If there are any advanced stats nerds/geeks/whoever out there who want to collaborate or make suggestions I would be thrilled to hear from you. I'd also happily welcome hearing from any other web developers who want to get involved. Email: jeff@hockeyvisualizer.com
